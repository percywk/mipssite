Journaling Nand Flash Memory System

The school project involved a creating a file management system for mock NAND gate flash memory. The biggest challenged occurred with the inability to reset individual bits of a memory sector. An entire sector could be reset to high, but once low an individual bit could only be reset by resetting the containing sector. My team, decided to forego other solutions for maintaining a journal of edits that would detail actions that did not need to be immediately reflected in the system. Instead metadata was stored that allowed files to be reconstructed. For the sake of brevity I will cut this short.
